# SigmaShake Homebrew Tap

Install [`ssg`](https://github.com/sigmashakeinc/ssg) — AI Agent Governance CLI:

```bash
brew install sigmashakeinc/tap/ssg
```

Or tap explicitly first:

```bash
brew tap sigmashakeinc/tap
brew install ssg
```

## Updating

When a new `ssg` version ships, the formula here is regenerated by
`bash sigmashake-dist/homebrew/update-formula.sh` in the
[main monorepo](https://github.com/sigmashakeinc/sigmashake-mono) and
copied into `Formula/ssg.rb` here.

## Verify

```bash
ssg --version
ssg --help
```
